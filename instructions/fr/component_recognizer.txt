
Sur cette page se trouve une information concernant un composant destiné à l’assemblage d’un ordinateur.
Ta tâche est d’identifier les blocs suivants sur la page :

1. Nom du composant
2. Description du composant
3. Caractéristiques du composant
4. Image du composant (si disponible)

Tu dois retourner un objet JSON avec les champs suivants :

```json
{
  "title": "nom du composant",
  "description": "description du composant",
  "specification": "caractéristiques du composant",
  "image_local_saved_path": "chemin local de l’image du composant, si disponible, sinon une chaîne vide"
}
```
